# Airdrop Points

Earn points through trading, creating tokens, and referring friends. Points will be used for future airdrop distributions.

## How to Earn Points

### Trading Volume
Earn **10 points for every $1** in trading volume.

### Deploy Token
Earn **1 point** for every token you create.

### Graduation Bonus
When a token graduates to PancakeSwap, bonus points are awarded:

| Rank | Points |
|------|--------|
| Token Creator | 5,000 |
| #1 Holder | 1,500 |
| #2 Holder | 1,000 |
| #3 Holder | 750 |
| #4-5 Holders | 500 each |
| #6-10 Holders | 250 each |

### Referral Points
Earn **5%** of points earned by users you referred.

## Leaderboard

The Airdrop page shows a real-time leaderboard ranking all users by total points. Track your progress and compete for the top spots.

## Season 1

The current airdrop season is **Season 1**. Details about token distribution and snapshot dates will be announced.
